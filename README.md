# Local Knowledge

Tampa Bay fishing intelligence, daily.

## Live Site

https://billroyle.github.io/local-knowledge-site

## About

Local Knowledge delivers daily fishing forecasts for Tampa Bay anglers — not generic weather, but actual conditions that matter.

## Features

- Daily 3-day forecasts for 5 Tampa Bay spots
- Star-rated conditions (1-5)
- "The Call" — clear fish/don't fish recommendation
- 4-star day SMS alerts
